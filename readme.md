# 🎲 Modern Dice Roller

A sleek, responsive, and interactive web-based dice rolling application. Built using clean HTML5, modern CSS3 (including Glassmorphism effects and animations), and vanilla JavaScript.

## 🚀 Features

* **Glassmorphism UI:** Features a modern, semi-transparent frosted-glass container with vibrant gradient backgrounds.
* **Smooth Animations:** Includes a dynamic shaking effect when the dice is rolled.
* **Fully Responsive:** Optimized for all screen sizes, including ultra-small mobile displays and landscape orientations.
* **Vanilla JavaScript:** Fast execution with zero dependencies or external frameworks.

## 🛠️ Technology Stack

* **HTML5:** Semantic markup structure.
* **CSS3:** Custom `@keyframes` animations, Flexbox layout, media queries, and linear gradients.
* **JavaScript (ES6):** State management, class manipulation, and random generation.
